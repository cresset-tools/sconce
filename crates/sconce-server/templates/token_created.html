Install in Composer

Add the repository, store the token (the token is the password — the username is ignored), then require a package:

composer config repositories.{{ repo }} composer {{ base }}/{{ org }}/{{ repo }}
composer config --auth http-basic.{{ host }} token {{ tok }}
composer require <vendor/package>

The auth line writes to auth.json. Use --global to reuse the token across projects, or set the COMPOSER_AUTH env var in CI instead of committing it.
